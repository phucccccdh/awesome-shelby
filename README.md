# Awesome Shelby

Community-driven resources for the Shelby ecosystem.

Shelby is building verifiable global object storage for AI workloads.

This repository tracks research, tools, and ecosystem developments.

---

## What is Shelby

Shelby is a decentralized data layer designed for AI workloads.

It aims to provide:

- verifiable data storage
- scalable object storage
- decentralized data infrastructure for AI
